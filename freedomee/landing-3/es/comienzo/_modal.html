¿Qué es un "token" o criptoactivo?

Antes de saber qué es un "token" o criptoactivo, es necesario entender qué es la criptomoneda. Aunque este término se utiliza ampliamente para describir cualquier moneda digital, existen marcadas diferencias entre ellas. Las características que se indican a continuación le ayudarán a entender qué es el token:

  • Se registran en la base de datos pública de otra criptomoneda;
  • Cualquier activo del mundo real, tangible o intangible, puede registrarse como toke;
  • Pueden o no servir como moneda de cambio;
  • Tienen una gran variedad de funciones potenciales, desde descuentos exclusivos, acceso a sistemas o derechos de voto;
  • Así, un token que representa una cantidad de oro físico, por ejemplo, puede ser intercambiado libremente entre sus usuarios. Así, sin posibilidad de intervención por parte de empresas o gobiernos.

¿Para qué sirven “tokens”?

Un token se utiliza para representar un activo en forma digital utilizando la tecnología blockchain, un sistema que registra las transacciones. Este registro es público, permanente y no censurable
